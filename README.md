# Meu Portfólio Pessoal

Este é o meu portfólio desenvolvido para apresentar meus projetos, habilidades e trajetória como desenvolvedor. O foco principal foi criar uma experiência interativa e performática.

---

## Tecnologias Utilizadas

* **React + Vite**: Base do projeto para alta performance.
* **Three.js / React Three Fiber**: Para renderização de elementos 3D interativos.
* **GSAP (GreenSock)**: Responsável pelas animações fluidas e efeitos de scroll.
* **TypeScript**: Garantindo maior segurança e escalabilidade no código.
* **CSS Modules / Tailwind**: Estilização moderna e responsiva.

---

## Funcionalidades

* **Animações de Scroll**: Efeitos visuais dinâmicos conforme o usuário navega.
* **Design Responsivo**: Adaptado para dispositivos móveis e desktop.
* **Seção de Projetos**: Vitrine com os principais trabalhos realizados.
